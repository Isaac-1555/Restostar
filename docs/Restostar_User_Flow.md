# User Flow – Restostar

## Restaurant Owner Flow
1. Visit Restostar website
2. Sign up / Login (Clerk)
3. Onboarding:
   - Owner & restaurant details
   - Upload logo
   - Google Maps review URL
   - Configure coupons
   - Choose email tone (AI/manual)
4. Access dashboard
5. View reviews by date
6. Open AI Analyzer tab

## Customer Flow
1. Scan restaurant QR code
2. Rate experience (1–5 stars)
3. If 4–5 stars:
   - Optional email entry
   - Click button → Google Maps review
4. If ≤3 stars:
   - Private feedback textarea
   - Optional email entry
   - Submit feedback
5. Receive coupon email
6. Redeem coupon in restaurant (single-use)
